# Task 2 - Sentiment Analysis with NLP

## Objective

Perform sentiment analysis on customer reviews using TF-IDF Vectorization and Logistic Regression.

## Tools Used

- Python
- Pandas
- Scikit-Learn
- Jupyter Notebook

## Steps Performed

1. Imported libraries
2. Created review dataset
3. Applied TF-IDF vectorization
4. Split training and testing data
5. Trained Logistic Regression model
6. Evaluated performance
7. Generated predictions

## Deliverables

- Sentiment classification
- Accuracy score
- Confusion matrix
- Classification report
